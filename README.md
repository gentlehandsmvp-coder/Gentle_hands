# 🌿 Gentle Hands Zambia - Complete Application

> *Professional in-home care delivered with kindness and dignity*

---

## 📖 Overview

Gentle Hands Zambia is a dual-sided marketplace connecting verified healthcare professionals with clients needing in-home care services. Built with a warm, compassionate brand identity that reflects trust, healing, and professional medical care in the Zambian context.
